Video 72. Generar build de producción - 05:25
Para poder hacer un build de producción de una aplicación que hayamos hecho, en este caso, lo que va del curso. Se utiliza el script en terminal:

npm build 

o

ng build —prod

Lo que hará una carpeta “build” en nuestro directorio y todos los archivos dentro de el es parte de nuestra aplicación a desplegar en un hosting.
